Privacy Policy

Briefly — Speech ⇄ Text Chrome Extension · Last updated 2026-05-19

The short version: Briefly does not run any server. Your audio, text, and API keys live only on your device, and audio is only ever sent to the speech provider you select. We never see, log, or store any of your data.

What Briefly does

Briefly is a Chrome extension that captures microphone audio in your browser and converts it to text (or vice-versa) using a speech provider of your choice (Web Speech API, AssemblyAI, OpenAI, Groq, Deepgram, or ElevenLabs).

What we collect

Nothing. Briefly has no backend server, no analytics, no telemetry, no crash reporting, and no advertising. The author of this extension does not receive any data from your use of it.

What is stored locally

The following data is stored on your device via Chrome's storage.local API and never leaves it except as described below:

  • API keys you paste in Settings (AssemblyAI, ElevenLabs, OpenAI, Groq, Deepgram).
  • User preferences (selected provider, voice, language, sliders).
  • History — up to the last 50 transcripts and TTS texts. You can clear this at any time from the history drawer or Settings → Reset all.

What is transmitted off your device

  • Audio you record is sent only to the speech-to-text provider you selected at the time of recording.
  • Text you type into the TTS box is sent only to the text-to-speech provider you selected.
  • Your API key for the selected provider is sent in the HTTP request to that provider so it can authenticate you.

Each provider has its own privacy policy. Please review them before use:

  • AssemblyAI: assemblyai.com/legal/privacy-policy
  • OpenAI: openai.com/policies/privacy-policy
  • Groq: groq.com/privacy-policy
  • Deepgram: deepgram.com/privacy
  • ElevenLabs: elevenlabs.io/privacy
  • Web Speech API (Google): handled inside the Chrome browser per Google's privacy policy.

Microphone access

Briefly requests microphone access only while you are actively recording. The audio stream is captured locally, processed in an offscreen document inside the extension, and either consumed by the browser's built-in Web Speech API or forwarded to the cloud provider you selected. The microphone stream is stopped the moment you press Stop or auto-stop fires.

Cookies, tracking, third parties

Briefly does not set cookies, does not use tracking pixels, and does not integrate with any analytics or advertising service.

Children

Briefly is not directed at children under 13 and does not knowingly collect any data from anyone.
